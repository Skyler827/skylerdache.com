Food Ordering System

Link Source

The food ordering system allows people to enter orders.

It was based off my expereince using the food ordering system at Buffalo Wild Wings as a server. Most of the menu items, ingredients, and prices are based off of their menu as well.

Features

  • Read configuration files defining dining rooms
  • Read configuration files defining menu items and categories
  • User Accounts and registration
  • Signing in as a server
  • Viewing restaraunt dining rooms
  • Seeing which dining rooms are accesable to the logged in user (green), which are assigned to others (purple), and which tables are unoccupied (white)
  • Browsing categories
  • Entering food and drink orders
  • Keep track of ingredient inventory
