Pong App

Link (coming soon) Source

The Pong app is a simple two player pong game with a 3D animated ball and pong paddles, controlled vis the up and down arrow keys.

I wrote this app to practice using websockets, 3D animation with three.js, and because I thought it would help me appreciate the complexity that goes into even simple games.

Features

  • Users may register accounts and login
  • User accounts contain a username, email address, created/updated dates and times, and a password
  • When a user logs in, they may start a pong game lobby
  • When a second user connects, the can join the lobby and play begins
  • Pong is displayed in a 3D environment
  • Players can control their pong paddle with the up and down arrow keys
  • Background music can be played or stopped independently by either player
  • The ball starts slow but goes faster as play continues
  • Each player starts with ten points and play ends when either player goes down to 0 points

Technical details

  • Writen in Java
  • Uses the Spring web framework with Spring Boot
  • Uses JPA to synchronize database state
  • Uses MariaDB (based on MySQL) to store data
  • Uses bcrypt to salt and hash account passwords
  • Uses websockets to synchronize game state between browser and server
  • Uses angular on the browser
  • Uses three.js to create 3D game envrionment
