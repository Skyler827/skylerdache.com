Pong App

Link (coming soon) Source

The Pong app is a simple two player pong game with a 3D animated ball and pong paddles, controlled vis the up and down arrow keys.

I wrote this app to practice my skills with real time data and 3d graphics, and thought it might be fun to play with a game I wrote.

Features

  • Allow registration and login
  • Allow a user to create and join a game
  • Displays a 3d pong scene
  • Send ball and paddle position information from server to browser using websockets
  • Keyboard control data to server in real time
  • Have a play/pause function
  • Have a background music start/stop button
