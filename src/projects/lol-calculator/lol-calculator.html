League of Legends Calculator

Link Source

The League of Legends Calculator calculates who would win in a fight between two selected League of Legends characters, with given levels and items.

I wrote this web app because the game of League requires constantly knowing if you can win a fight against a given opponent, because if you can win, you should pursue, and if you can't win, you should try to escape.

This app doesn't solve the problem of optimal play in general (yet), but it does the following:

Features

  • On app startup, fetch data from offical Riot game servers on all characters, items, and combat statistics
  • Record champion data for easy querying later
  • Present a responsive base interface
  • Allow selecting opposing champions
  • Allow selecting champion levels
  • Allow selecting up to 6 items for each champion
  • Calculates the result of a fight consisting of basic attacks
  • Incorporates champion health, attack damage, and armor
  • Displays a line graph of character's health over time
  • Displays the name of the winning character

Technical Details

  • Written in Python
  • Currently set up to run within the Django framework, but currently does not use any Django features
  • On previous iterations, used the Flask web framework
  • Uses SQLite to store basic character data
  • Uses jQuery for DOM manipulation in the browser
  • Images, after being queried during server startup, are stored on disk
  • Runs under gunicorn as a systemd service on Ubuntu Linux
  • Deployed behind Nginx reverse proxy, which serves static data as needed
